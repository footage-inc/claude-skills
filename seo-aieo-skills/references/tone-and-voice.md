# トーン＆ボイス ガイドライン

## Footageのブランドボイス

**基本姿勢**: 専門家として正確な情報を提供しつつ、読者（患者・家族・ケアマネ・開業検討者）に寄り添う温かさを維持する。

## カテゴリ別トーン

### 訪問看護集客（toC: 患者・家族向け）
- **一人称**: 「私たち」「私たち訪問看護師」
- **読者への呼びかけ**: 「あなた」「ご家族」「患者様」
- **文末**: 「〜できます」「〜をお伝えします」「〜をサポートします」
- **避ける表現**: 「〜すべきです」「〜しなければなりません」→ 代わりに「〜できると安心です」「〜がおすすめです」

### 経営支援集客（toB: 開業者・経営者向け）
- **一人称**: 「私たち」「弊社」
- **読者への呼びかけ**: 「経営者の方」「開業を検討されている方」
- **文末**: 「〜が重要です」「〜をお勧めします」
- **避ける表現**: 上から目線の断定 → 代わりに「〜という視点が有効です」「〜の事例があります」

### デイサービス集客（toC: 患者・家族向け）
- **一人称**: 「私たち」「Re-move」
- **読者への呼びかけ**: 「ご本人」「ご家族」
- **文末**: エビデンスに基づく提案 + 共感
- **特徴**: 科学的根拠 → 具体的プログラム → 感情的結び の三段構成

## NG表現集

| NG | OK | 理由 |
|---|---|---|
| 〜すべきです | 〜できると安心です | 命令的 → 提案的 |
| 必ず〜してください | 〜をお勧めしています | 強制感 → 寄り添い |
| 〜は危険です | 〜には注意が必要です | 恐怖訴求 → 情報提供 |
| 当社は〜 | 私たちは〜 | 距離感 → 親近感 |
| 〜に決まっています | 〜と考えられています | 断定 → エビデンスベース |

## 文章の長さ・リズム

- 一文は60文字以内を目安。長文は2文に分割
- 段落は3〜5文。それ以上は小見出しで区切る
- 専門用語には初出時に括弧で平易な説明を添える
  - 例: 「オートファジー（細胞の自己修復機能）」
