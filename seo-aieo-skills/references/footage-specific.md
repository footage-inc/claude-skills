# Footage固有ルール

## 会社概要

- **社名**: 株式会社Footage（フッテージ）
- **サイト**: footage-nursing.jp
- **事業**: 訪問看護ステーション、デイサービス（Re-move）
- **エリア**: 名古屋市を中心とした東海圏
- **デイサービス**: Re-move（リムーブ）— 名古屋市昭和区、パーキンソン病および運動障害特化型
- **デイサービスサイト**: https://fitness-remove.jp/
- **訪問看護サイト**: https://footage-nursing.jp/

## 差別化ポイント（記事に盛り込む要素）

### 訪問看護事業
1. **多職種チーム体制**: 看護師、理学療法士、作業療法士、言語聴覚士がワンチームで対応
2. **DXツール活用**: バイタル推移のリアルタイム同期、チーム内情報共有
3. **24時間対応**: 夜間の緊急連絡時も経過を把握した状態で対応
4. **臨床推論**: 「なぜその症状が出ているか」を論理的に分析するアプローチ

### デイサービス事業（Re-move）
1. **PD特化型プログラム**: LSVT-BIG、RAS、デュアルタスク訓練
2. **個別設計**: ヤール度に合わせたプログラム調整
3. **専門スタッフ**: PT/OT常駐、オン・オフ現象への対応力
4. **実績**: 体験利用当日の成約率90%（2026年2月現在）

### 経営支援事業
1. **自社実績ベースのノウハウ**: 理論ではなく実践から得た知見
2. **開業支援**: ステーション立ち上げから運営安定化まで
3. **採用・定着**: オープニングスタッフ採用、離職防止の仕組み
4. **自律分散型組織**: 訪問看護×ティール組織の独自モデル（5本シリーズ公開済み）
5. **臨床推論文化**: スタッフ全員が「なぜ」を考える学習環境
6. **経営と現場の一致**: 対立構造をどう解消したかの実体験
7. **黒字倒産リスクへの対処**: 増収が招く資金ショートの実例と対策

## 経営支援記事テーマ設計ガイド（独自性確保）

### テーマ提案の3軸構造

テーマを提案する際、以下の3軸を必ず組み合わせる。単なるトピック名だけの提案は禁止。

```
テーマ = トピック × 独自アングル × Footage実体験
```

**例:**
- NG: 「訪問看護ステーションのICT活用法」 ← 誰でも書ける一般論
- OK: 「バイタルの"見える化"が変えた訪問看護の現場｜DXで離職率が下がった理由」 ← Footage実体験 × 逆説的洞察

### 独自アングルのパターン

| パターン | 説明 | 例 |
|---|---|---|
| **逆説・反常識** | 通説と逆のことを主張 | 「増収が招く資金不足の正体」 |
| **失敗→学び** | 自社の失敗経験から得た教訓 | 「開業1年目に犯した3つの判断ミス」 |
| **構造的洞察** | 業界構造の問題を言語化 | 「経営と現場の対立はなぜ起きるのか」 |
| **哲学・思想** | Footageの経営哲学を展開 | 「自律分散型組織×訪問看護の親和性」 |
| **数字・データ** | 自社データで語る | 「訪問件数と離職率の相関を分析して見えたこと」 |
| **比較・対照** | 他業界の手法を訪問看護に適用 | 「SaaS企業のオンボーディングを訪問看護に導入した結果」 |
| **学術×実践** | 経営学・組織論の理論を訪問看護に接続 | 「ドラッカーの"知識労働者"論から考える訪問看護マネジメント」 |

### 学術的深みの持たせ方

経営支援記事は読者が経営者・管理者のため、学術的な裏付けがあると説得力と権威性が増す。以下のフレームワークや理論を記事テーマに組み込む。

**コア哲学（CEO思想基盤）:**
- スピノザ: コナトゥス（自己保存の力）→ スタッフの内発的動機づけ
- フーコー: 系譜学・権力と知の関係・生政治 → 組織における管理と自律の緊張関係、ケアの権力構造
- フランクル: 意味への意志・ロゴセラピー → 訪問看護師の仕事の意味
- ヒューム: 経験主義・道徳感情論・共感（sympathy）の理論 → ケアの本質、チーム内の共感力
- 唯識思想: 自己認識の変容 → 「自責」文化、自己変革のプロセス
- モリス・バーマン: 身体性の知・再魔術化 → 触れるケア、身体知の重要性
- ジョナサン・ハイト: 道徳基盤理論・象と騎手 → 組織の価値観設計、直感と理性のマネジメント
- 國分功一郎: 中動態の世界・暇と退屈の倫理学 → 「する/される」を超えたケアの主体性、意志と行為の再考
- ウィトゲンシュタイン: 言語ゲーム・家族的類似性 → 暗黙知の言語化の限界、現場の「ルール」の本質
- ハンナ・アーレント: 活動・公共性・悪の凡庸さ → 「考えることの放棄」が現場にもたらす危険、判断力と主体性
- 清水哲郎: 臨床倫理・意思決定プロセス → ACP・終末期の意思決定支援、本人の意向を尊重するケアの学術基盤

**コア経営理論:**
- バーニー（RBV: 資源ベースの経営戦略）: ステーションの内部資源（人材・文化）が競争優位の源泉
- 野中郁次郎（知識創造理論・SECI モデル）: 暗黙知→形式知の変換 → 臨床推論文化・ナレッジマネジメント
- ポーター（競争戦略・バリューチェーン）: 差別化戦略としての専門特化
- オライリー&タッシュマン（両利きの経営）: 既存事業の深化と新領域の探索
- トム・ニクソン（ソース原理）: ソース・サブソース・クリエイティブフィールド → 組織の創造性の源泉、リーダーは「管理者」ではなく「ソース（源泉）」、権限委譲とサブソースの育成

**組織論・マネジメント:**
- ティール組織（フレデリック・ラルー）: Footageの自律分散型組織の理論的背景
- サーバントリーダーシップ（グリーンリーフ）: 管理者の役割転換
- 心理的安全性（エドモンドソン）: スタッフが意見を言える環境づくり
- センスメイキング（ワイク）: 不確実な在宅現場での意思決定
- マックス・ウェーバー: 官僚制・支配の正当性 → 「管理型組織」から「自律型組織」への転換の理論的背景
- ジリアン・テット: サイロ・エフェクト → 多職種連携を阻む縦割り構造の可視化と打破
- パットナム: ソーシャル・キャピタル → 地域連携・紹介ネットワークの信頼資本としての構築
- ケネス・ガーゲン: 社会構成主義・関係性の理論 → 対話が現実を作る、チーム内の意味共有プロセス

**人材開発・教育:**
- 中原淳: 経験学習・職場学習論 → スタッフの成長環境設計
- 那須正裕: 主体的・対話的な深い学び → 訪問看護の学習文化
- 松岡正剛: 編集工学・知の編集 → 多職種の知を統合する力
- ヴィゴツキー: 発達の最近接領域（ZPD）→ 新人育成の足場かけ（scaffolding）、経験者との協働学習

- 安宅和人: イシューからはじめよ・シン・ニホン → 問いの設計力、データ駆動型経営

**その他（社会・文化批評）:**
- イヴァン・イリイチ: 脱学校化・脱専門家化・コンヴィヴィアリティ → 医療の専門家支配への問い直し、利用者主体のケア

**認知科学・知の理論:**
- J.J.ギブソン: アフォーダンス理論 → 環境が行為を導く、在宅環境のケアデザイン
- マイケル・ポランニー: 暗黙知の次元 → 野中SECIモデルの源流、「語れる以上のことを知っている」
- レヴィ=ストロース: ブリコラージュ（器用仕事）→ 限られた資源で創造する訪問看護の実践知
- 鈴木宏昭: 認知科学・プロジェクション → 身体性認知、直感と判断の科学的理解

**医療・看護学:**
- エビデンスベースドプラクティス（EBP）: 臨床推論文化との接続
- 多職種連携（IPW: Interprofessional Work）: チーム医療の学術基盤
- レジリエンスエンジニアリング: 安全文化とエラー対応
- ナラティブベースドメディスン: 患者の語りを重視するケア
- ドナベディアン: 医療の質評価モデル（構造・過程・結果）→ ステーションの質管理フレームワーク

**使い方:**
- テーマ提案時に「このテーマはどの学術理論と接続できるか」を1つ以上明示する
- 記事本文では理論の引用→Footageの実践事例→読者への示唆の3段構成で展開
- 論文・書籍の出典を最低1つ含める（権威性+LMO対策）

### Footageが語れるテーマ資産（KBから抽出）

以下はFootageの実体験・社内文化に基づいた独自テーマ群。一般論では書けないもの。

**組織・文化系:**
- 自律分散型組織への移行プロセスと現場の反応
- 「自責」文化の導入で起きた変化
- スタッフの自発的学習をどう仕組み化したか
- 管理者の役割を「管理」から「支援」に変えた話

**経営リアル系:**
- 開業初年度のキャッシュフロー危機と乗り越え方
- 訪問件数を追いすぎた時期に起きたこと
- 「良い人材が辞める」本当の理由と対策
- 営業が苦手な看護師でも紹介が増える仕組み

**採用・定着系:**
- オープニングスタッフ採用で重視した「カルチャーフィット」
- 面接で聞くべき質問と聞いてはいけない質問
- 入職3ヶ月の離職をゼロにした仕組み

**連携・地域系:**
- ケアマネジャーが「また頼みたい」と思うステーションの共通点
- 病院との退院カンファレンスで信頼を勝ち取る方法
- 地域の多職種連携で「顔が見える関係」を作るまでにやったこと
