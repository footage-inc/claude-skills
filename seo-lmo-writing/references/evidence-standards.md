# エビデンス＆引用基準

## 基本原則

Footageの記事は「根拠のある専門情報」が差別化ポイント。感想文やまとめサイト的な記事にならないよう、医療・経営いずれの記事にもエビデンスを含める。

## 引用レベル

### Tier 1: 最も信頼度が高い（優先使用）
- 診療ガイドライン（日本神経学会、日本呼吸器学会等）
- Cochrane Review、メタアナリシス
- 厚労省・自治体の公式統計データ
- 介護保険制度の公式文書

### Tier 2: 補助的に使用
- PubMed掲載の査読付き論文（RCT、コホート研究）
- 学会発表のアブストラクト
- 業界団体の調査レポート（日本訪問看護財団等）

### Tier 3: 参考程度
- 書籍、教科書
- 専門メディアの解説記事
- 自社の実績データ（「体験利用当日の成約率90%」等）

## 記事内での表現ルール

### DO（推奨）
- 「〜というエビデンスがあります（参考: ○○学会ガイドライン2024）」
- 「○○大学の研究によると、〜と報告されています」
- 「厚労省のデータでは〜」
- 「当ステーションの実績として〜」（自社データは明示）

### DON'T（避ける）
- 「〜と言われています」（誰が？ → 出典を明記）
- 「研究で証明されています」（どの研究？ → 具体化）
- 「多くの方が〜」（何人？何%？ → 数値化）
- 「最新の研究では〜」（いつの？ → 年度を明記）

## カテゴリ別の必須エビデンス

### 訪問看護集客
- 疾患の説明 → ガイドラインまたは教科書レベルの正確性
- ケアの方法 → 看護プロトコルに準拠
- Footageの強み → 具体的な体制・ツール名

### 経営支援集客
- 市場データ → 厚労省統計、介護事業経営実態調査
- 経営指標 → 具体的な数値（利用者数、収支目安等）
- 成功事例 → Footageの実績を匿名化して提示

### デイサービス集客
- リハビリ手法 → 論文引用（RAS、LSVT-BIG等）
- 効果 → 研究結果の数値（歩幅○%改善等）
- Re-moveの実績 → 成約率等の自社データ
