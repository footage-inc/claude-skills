# 小児慢性特定疾病 在宅療養支援ガイド

## 1. 目的
小児慢性特定疾病（小慢）を抱える子どもと家族が、地域で安心して成長・発達できるよう、医療、福祉、教育が連携した支援を行うための指針です。

## 2. 支援の柱
- **医療的ケアの確立**: 吸引、経管栄養、人工呼吸器管理などを家族が安全に行えるよう指導・支援する。
- **発達・学習支援**: 病気を抱えながらも、園や学校での生活が送れるよう教育機関と連携する。
- **家族のQOL向上**: 親の介護負担軽減（レスパイトケア）や、きょうだい児への配慮を支援に含める。

## 3. 訪問看護における具体的実践
- **成長のアセスメント**: 単なる病状観察だけでなく、子どもの「発達段階」に合わせた遊びや学習の機会を検討する。
- **緊急時対応の構築**: 停電時のバッテリー確保や、急変時の連絡経路を家族と共にシミュレーションする。
- **自立（自律）の支援**: 子ども自身が自分の病気について理解し、生涯を通じて自己管理できるよう段階的に支援する（移行期医療への橋渡し）。

## 4. 移行期医療への支援
小児科から成人診療科へのスムーズな移行を支援する。特に、成人期特有の社会保障制度や就労支援についての情報提供を行う。

## 5. Footageの取り組み
- **シームレスな多職種連携**: 教育機関や相談支援事業所との情報共有にデジタルツールを活用。子どもの「学校での様子」と「家での様子」を繋ぎ、一貫した発達支援を可能にします。
