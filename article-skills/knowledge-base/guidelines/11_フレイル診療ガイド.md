# フレイル診療ガイド

## 1. 出典・エビデンスレベル

- **日本老年医学会「フレイル診療ガイド 2018」**
- **Fried Frailty Phenotype**（Fried et al., 2001, *J Gerontol*）
- **J-CHS基準**（Satake et al., 2016, *Geriatr Gerontol Int*）
- **厚生労働省 基本チェックリスト**（25項目）
- **Kojima et al. (2017, JAMDA)**: 日本のフレイル有病率メタ分析

## 2. フレイルの定義

加齢に伴う複数の生理的システムの予備能低下により、ストレスへの脆弱性が増大した状態。**適切な介入により可逆的**であることが最大の特徴。

### フレイルの3つの側面
- **身体的フレイル**: 筋力低下、歩行速度低下、体重減少
- **精神・心理的フレイル**: うつ、認知機能低下、意欲低下
- **社会的フレイル**: 閉じこもり、社会的孤立、経済的困窮

### フレイルサイクル（負のスパイラル）
サルコペニア → 活動量低下 → 食欲低下・低栄養 → さらなるサルコペニア → 要介護状態

## 3. 診断基準

### A. J-CHS基準（日本版）

| 項目 | 基準 |
|---|---|
| **体重減少** | 6ヶ月で2〜3kg以上の意図しない体重減少 |
| **筋力低下** | 握力：男性 < 28kg、女性 < 18kg |
| **疲労感** | 「わけもなく疲れたような感じがする」に該当 |
| **歩行速度低下** | < 1.0 m/s |
| **身体活動量低下** | 軽い運動・定期的な運動の両方に「いいえ」 |

- **0項目**: 健常（ロバスト）
- **1〜2項目**: プレフレイル
- **3項目以上**: フレイル

### B. 基本チェックリスト（厚生労働省・25項目）

| 領域 | 項目番号 | 該当閾値 |
|---|---|---|
| 日常生活関連動作（IADL） | No.1-5 | 3項目以上 |
| 運動機能 | No.6-10 | 3項目以上 |
| 栄養状態 | No.11-12 | 2項目 |
| 口腔機能 | No.13-15 | 2項目以上 |
| 閉じこもり | No.16-17 | No.16が「いいえ」 |
| 認知機能 | No.18-20 | 1項目以上 |
| うつ | No.21-25 | 2項目以上 |

- **総合スコア 7点以上**: フレイルに該当（Satake et al.で検証済み）

## 4. 疫学データ（日本）

- **フレイル有病率**: 地域在住高齢者の約 **7.4%**（Kojima et al., 2017メタ分析）
- **プレフレイル有病率**: 約 **48.1%**
- **年齢階層別**:
  - 65〜69歳: 3〜5%
  - 70〜74歳: 6〜8%
  - 75〜79歳: 10〜15%
  - 80〜84歳: 20〜25%
  - 85歳以上: 30〜40%
- 女性の方が男性より有病率が高い
- 日本は高齢化率29%超で、フレイル対策は主要な公衆衛生課題

## 5. エビデンスに基づく介入

### A. 運動介入（エビデンスレベル: A — 最も強固）

| 運動の種類 | 効果 | エビデンス |
|---|---|---|
| **レジスタンストレーニング** | 歩行速度・握力・ADL改善 | メタ分析で確認（Cadore 2013, de Labra 2015） |
| **複合運動プログラム** | フレイル改善に最も有効 | Cochrane Review支持 |
| **バランス訓練** | 転倒率30〜35%減少 | Otago Exercise Programme |
| **ウォーキング** | 週1回増で要介護リスク約20%低下 | 日本の大規模コホート |
| **デュアルタスク（コグニサイズ）** | 認知機能低下予防で運動単独より優位 | Doi et al. 2017 |

- 推奨頻度: 週2〜3回
- 効果発現: 最低12週間の継続が必要

### B. 栄養介入（エビデンスレベル: A〜B）

| 栄養素 | 推奨量 | エビデンス |
|---|---|---|
| **タンパク質** | 1.0〜1.2 g/kg/日（リスク者は1.2〜1.5） | PROT-AGE Study Group |
| **1食あたりのタンパク質** | 25〜30g（筋タンパク合成最大化） | 日本人の食事摂取基準2020 |
| **ビタミンD** | 800〜1000 IU/日（不足時） | 中等度エビデンス |
| **ロイシン** | 2.5〜3.0g/食 | mTORシグナル活性化 |

- 日本の伝統的食事は朝・昼のタンパク質が不足しがち
- **運動＋栄養の組み合わせ**がどちらか単独より有効（Ng et al., 2015 RCT, *JAMA Internal Medicine*）

### C. 社会参加（エビデンスレベル: B）

- 社会的孤立・閉じこもりはフレイル発症の独立したリスク因子
- **通いの場**（サロン、地域の集いの場）参加でフレイル発症率20〜30%低下（JAGES研究）
- 社会的フレイルが身体的フレイルに先行（Makizako et al., 2015, 2018）

## 6. デイサービスにおける実践的意義

### フレイルの3側面すべてに対応可能

| フレイルの側面 | デイサービスの対応 |
|---|---|
| **身体的** | 機能訓練（個別機能訓練加算I・II）、バランス・筋力・持久力トレーニング |
| **栄養** | 栄養改善加算による栄養評価・食事指導、昼食でのタンパク質確保（25g以上） |
| **口腔** | 口腔機能向上加算による嚥下訓練・口腔ケア |
| **社会的** | 定期的な外出機会、仲間との交流、グループ活動 |
| **認知** | デュアルタスクトレーニング、認知刺激活動 |

### 成果指標
- J-CHS基準の改善（フレイル→プレフレイル→健常）
- 歩行速度（目標: 1.0 m/s以上）
- 握力（目標: 男性28kg以上、女性18kg以上）
- 基本チェックリストスコアの改善
- LIFE（科学的介護情報システム）への登録・報告

## 7. Footageの強み

- リムーブの複合運動プログラム（レジスタンス＋バランス＋有酸素）でフレイルサイクルを断ち切る
- PT/OTによるJ-CHS基準に基づいた個別評価と目標設定
- 栄養面でもタンパク質摂取量を意識した食事提供
- 定期的な通所による社会参加の確保で、閉じこもり・社会的フレイルを予防
