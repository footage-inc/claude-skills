# 介護保険と医療保険の訪問看護の違いは？

訪問看護を利用する際、最も多くの方が戸惑うのが「介護保険」と「医療保険」の使い分けです。ご本人の年齢や疾患によって、どちらの保険が適用されるか（一方が優先されるか）が法律で決まっています。

この記事では、制度の違いを整理し、Footage（フッテージ）がどのようにスムーズな導入をサポートしているかをお伝えします。

## どちらの保険が優先される？

原則として、**「介護保険」の対象者は介護保険が優先**されます。しかし、特定の疾患がある場合や病状が悪化した場合には「医療保険」に切り替わります。

### 1. 介護保険が適用されるケース
- 65歳以上で、要支援・要介護認定を受けている方
- 40歳〜64歳で、16種類の特定疾病（末期がん、関節リウマチ等）により要介護認定を受けている方

### 2. 医療保険が適用されるケース
- 40歳未満の方（小児など）
- 40歳〜64歳で、介護保険の要介護認定を受けていない方、または特定疾病以外の方
- **【例外】** 介護保険の対象者であっても、以下の場合は医療保険となります。
  - 厚生労働大臣が定める疾病（難病、末期がん、人工呼吸器使用など = 「別表第7」）
  - 急性増悪等により、一時的に頻繁な訪問が必要と主治医が判断した場合（特別訪問看護指示の発行）
  <!-- 修正：精神科訪問看護が必要な方も含める -->

## 制度の主な違い一覧

| 比較項目 | 介護保険 | 医療保険 |
| :--- | :--- | :--- |
| **利用回数の制限** | ケアプランの範囲内で制限なし | 原則週3回まで（※難病等は毎日可） |
| **1回の訪問時間** | 20分、30分、60分、90分など | 30分〜90分程度 |
| **自己負担割合** | 原則1〜3割 | 原則1〜3割（公費助成等あり） |
| **相談窓口** | ケアマネジャー | 訪問看護ステーション・主治医 |

## Footageでの対応：複雑な制度もチームでサポート

Footageでは、事務スタッフと看護師が連携し、複雑な保険適用の判断を迅速に行います。

- **DXを活用した即時確認**: 医師からの指示内容や利用者の状態をプラットフォームで共有。介護から医療への切り替えが必要な際も、タイムラグなく主治医と調整を行います。
- **「制度」ではなく「利用者」に合わせる**: 私たちは制度の枠に縛られるのではなく、「目の前の利用者様に何が必要か」を使命感を持って判断し、最適なプランを提案します。
- **公費負担の案内**: 難病や障害など、医療費の助成制度が利用できる場合は、申請のアドバイスも積極的に行い、ご家族の負担軽減を図ります。

## まとめ

介護保険と医療保険の仕組みを完璧に理解する必要はありません。私たちは、制度の橋渡し役として、利用者様が常に最適な条件で看護を受けられるよう並走します。
