# 記事の目的・ゴール

## 全体目的
株式会社Footageの事業成長を支えるコンテンツマーケティング。
SEO記事を通じて見込み客を集客し、各事業への問い合わせ・応募を増やす。

### 上限到達後の運用
上限に達したテーマは**新規作成を停止**し、以下の**分析ベースのリライト**に移行する：
1. **パフォーマンス分析**: 検索順位、CTR、直帰率、CV数をもとに改善対象記事を特定
2. **リライト優先度**: 「検索流入があるがCVが低い記事」を最優先
3. **リライト内容**: SEOキーワードの最適化、構成の再編、CTA改善、最新情報への更新
4. **頻度**: 上限到達後も同じ週次ペース（訪問看護2本/週、経営支援2本/週、デイサービス1本/週）でリライトを実施

## テーマ別ゴール

### 訪問看護集客（column）
- **目的**: 訪問看護サービスの利用者獲得・認知拡大
- **KPI**: 検索流入数、問い合わせ数
- **記事のゴール**: 読者が「Footageの訪問看護に相談してみよう」と思うこと
- **CTA**: 訪問看護の相談・お問い合わせ
- **URL形式**: `https://footage-nursing.jp/column/{ID}/`

### デイサービス集客（column）
- **目的**: Re-move（リハビリ特化型デイサービス）の利用者獲得
- **KPI**: 検索流入数、見学・体験申込数
- **記事のゴール**: 読者が「専門的なリハビリを受けられる場所がある」と認識すること
- **CTA**: デイサービスの見学・体験申込
- **URL形式**: `https://footage-nursing.jp/column/{ID}/`
- **注力テーマ（2026Q1〜）**:
  - パーキンソン病（10記事・上限到達）
  - フレイル予防（GL: 11_フレイル診療ガイド）
  - サルコペニア予防（GL: 12_サルコペニア診療ガイドライン）
  - 変形性膝関節症・膝の痛み（GL: 13_変形性膝関節症診療ガイドライン）
  - 腰痛・腰痛改善（GL: 14_腰痛診療ガイドライン）
  - 大腿骨頚部骨折後のリハビリ（GL: 15_大腿骨頚部骨折診療ガイドライン）
  - 介護予防・重症化予防（GL: 16_介護予防・重症化予防ガイドライン）
  - 活動低下・運動不足・廃用症候群（GL: 16_介護予防・重症化予防ガイドライン）

### 採用マーケ（column）
- **目的**: 訪問看護師の採用応募数増加
- **KPI**: 検索流入数、採用ページ遷移数、応募数
- **記事のゴール**: 読者が「Footageで働いてみたい」と思うこと
- **CTA**: 採用情報ページ・エントリーフォーム
- **URL形式**: `https://footage-nursing.jp/column/{ID}/`

### 経営支援集客（rec_column）
- **目的**: FCフランチャイズ加盟・開業支援の問い合わせ獲得
- **KPI**: 検索流入数、資料請求数、説明会申込数
- **記事のゴール**: 読者が「Footageに開設相談をしてみよう」と思うこと
- **CTA**: FC説明会・資料請求・開業相談
- **URL形式**: `https://footage-nursing.jp/opening/column/{ID}/`
