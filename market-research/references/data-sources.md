## 補完データソース1: JMAP 地域医療情報システム（施設個別データ）

### 概要

| 項目 | 内容 |
|---|---|
| 正式名称 | 地域医療情報システム（JMAP） |
| 運営元 | 公益社団法人 日本医師会 |
| URL | https://jmap.jp/ |
| 費用 | 無料（統計DLは医師会員限定） |
| データソース | ウェルネスデータベース（毎月更新）、総務省統計局、社人研推計、厚労省 |
| 引用例 | `出典：日本医師会 地域医療情報システム` |

### 信頼性評価: A（高信頼）

理由:
- 日本医師会が運営する公式システム
- データソースはウェルネスデータベース（月次更新）+ 政府統計
- 医療機関情報は地方厚生局・都道府県の公開情報に基づく

### JMAPの強み（医療需給総覧にないデータ）

| データ | JMAP | 医療需給総覧 |
|---|---|---|
| 訪問看護ステーション数・看護師配置 | **あり** | なし |
| 介護事業所数（訪問型/通所型/入所型） | **あり** | なし |
| 在宅療養支援診療所・病院の数 | **あり** | 在後病指定のみ |
| 施設ごとの所在地・地図 | **あり** | 病床地図のみ |
| 医療介護需要予測指数 | **あり**（2050年まで） | なし（独自推計あり） |
| 介護職員数（常勤換算） | **あり** | なし |

### URL構造

```
# 二次医療圏の統計ページ（WebFetchで取得可能・最重要）
https://jmap.jp/cities/detail/medical_area/{医療圏コード}
例: https://jmap.jp/cities/detail/medical_area/2708

# 都道府県の統計ページ
https://jmap.jp/cities/detail/prefect/{都道府県コード}
例: https://jmap.jp/cities/detail/prefect/27

# 施設詳細ページ（WebFetchで取得可能）
https://jmap.jp/facilities/detail/{施設コード}
例: https://jmap.jp/facilities/detail/2761090113-05

# 施設検索結果（部分的に取得可能）
https://jmap.jp/facilities/search/type:0/searchArea:medical_area/searchId:{医療圏コード}
```

### 統計ページで取得できるデータ項目

**医療機関**:
- 一般診療所数（診療科系別8分類）、病院数、歯科数、薬局数
- 在宅療養支援診療所数、在宅療養支援病院数
- 訪問歯科数、訪問薬局数
- 病床数（一般/療養/精神/結核・感染症）
- 医師数・歯科医師数・薬剤師数（常勤）

**介護施設**:
- 介護施設合計数
- 訪問型施設数（訪問介護・訪問看護・訪問リハ等の合計）
- 通所型施設数
- 入所型施設数
- 特定施設数
- 居宅支援事業所数
- 福祉用具事業所数
- 介護職員数（常勤換算）

**人口・将来推計**:
- 現在人口、高齢化率、人口密度
- 将来推計人口（年齢4区分、2050年まで）
- 医療介護需要予測指数（2020=100基準）

**注意**: 訪問型施設数は「訪問介護+訪問看護+訪問リハ等」の合計値。訪問看護ステーション単独の件数は統計ページでは不明。施設検索で「訪問看護」フィルタをかけて件数を確認する必要がある。

## 競合ツールとの使い分け

| ツール | 強み | 弱み | 使いどき |
|---|---|---|---|
| **医療需給総覧** | 分析済みPDF、将来推計2050年まで、病院個別データ | PDF形式で加工困難、訪問看護ST未掲載 | メインの分析ソース |
