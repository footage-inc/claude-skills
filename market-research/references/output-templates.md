# 出力テンプレート

## 引用ルール（必須）

### 医療需給総覧のデータを引用する場合

```
出典：医療需給総覧 ○○県△△医療圏（株式会社日本経営）
```

### 複数ソースを組み合わせた分析の場合

```
出典：医療需給総覧 ○○県△△医療圏（株式会社日本経営）をもとに作成
```

### 将来推計値を記載する場合

必ず「推計」「見込み」の文言を付ける。確定値として扱わない。

```
OK: 2050年の在宅医療患者数は約22,800件/月と推計される
NG: 2050年の在宅医療患者数は22,806件/月である
```
