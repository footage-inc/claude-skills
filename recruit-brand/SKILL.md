# FOOTAGE 採用ブランドスキル

FOOTAGE採用ブランドナレッジベース（KB v0.3.1）に基づき、採用関連のあらゆるコンテンツ・企画・戦略を設計・制作するスキル。

## 最初に必ずやること

**どのタスクでも、作業開始前にKBを読み込む。**

```
/Users/yusuke/footage-aix/knowledge-base/recruit/brand/recruit-brand-kb-v031.md
```

KBを読まずに採用コンテンツを作ると、ブランドの一貫性が崩れる。これがこのスキルの核心。

## タスク別フロー

### A. 採用コンテンツ制作（SNS投稿・広告・求人票等）

1. **KB読み込み**（ルーティングテーブルに従い該当セクションを重点参照）
2. **ペルソナ特定**: §7 のどのペルソナ向けか確認
3. **制作**:
   - §4 ブランドボイスのトーン原則を必ず適用
   - §1 EVPの3つの柱（自律・透明・成長）のうち、どれを訴求するか意識
   - §11 ファクトシートから具体的な数字を引用
4. **セルフチェック**:
   - §6 アンチパターンの全項目に照合（1つでも該当したら修正）
   - §4 のGood/Bad例と比較してトーンを確認
5. **出力**

### B. 採用イベント企画（コミュニティナイト等）

1. **KB読み込み**（§0, §1, §2, §4, §6, §7, §9 を重点参照）
2. **目的整理**: §0 存在意義・§1 EVPとの接続を明確化
3. **企画設計**:
   - §2 カルチャーが「体験」として伝わる設計にする
   - §7 ペルソナ別のアプローチを含める
   - §9 ビジュアルアイデンティティ（カラー含む）を適用
4. **チェック**:
   - §6 アンチパターンに抵触しないか
   - §4 ブランドボイスのNG表現を使っていないか
5. **出力**

### C. 採用戦略・ブランディング議論

1. **KB全体を読み込み**
2. **§13 戦略ブレストインサイト**の6つの主要施策を前提に議論
3. **§5 採用選考基準**との整合性を確認（コンテンツで惹きつける人 = 選考で評価する人）
4. **提案・議論**

## 絶対に守ること

1. **§6 アンチパターンは不可侵** — 「誰でも歓迎」「急募」「夢のような職場」は絶対NG
2. **§4 ブランドボイスが全コンテンツの品質基準** — 経験ベースで語る、思想の背景を伝える、問いかけを残す
3. **数字は§11 ファクトシートから引用** — 根拠なき数字や曖昧な表現は使わない
4. **§9-A ブランドカラーパレット準拠** — Orange(`#F08E21`), Navy(`#2E3A62`), Off-White(`#F2F4F1`)
5. **「ENJOY with FOOTAGE」はブランドプロミス** — 軽々しく使わず、§0-1の定義を理解した上で使用する
