# 仮説駆動分析プロトコル

> KPI変動の原因分析時に必ず従うプロトコル。最初にもっともらしい仮説が見つかっても、必ず3つ以上の代替仮説を検証してから結論を出す。

## Step 1: 事実確認

**数字を見る。解釈はしない。**

以下のテンプレートで事実を整理する:

```
| 指標 | 今週 | 前週 | 前週比 | 目標値 | 目標比 | 前年同期 |
|------|------|------|--------|--------|--------|----------|
| CPA  |      |      |        |        |        |          |
| CVR  |      |      |        |        |        |          |
| CTR  |      |      |        |        |        |          |
| CPC  |      |      |        |        |        |          |
| 応募数|      |      |        |        |        |          |
| 費用  |      |      |        |        |        |          |
```

- 前年同期データがない場合は空欄とし、季節カレンダーで補完
- 学習期間中のデータには「⚠️」マークを付ける
- **この段階では「なぜ」を考えない**。まず全体像を把握する

## Step 2: 仮説の発散

**最低3つの仮説を列挙する。ここで絞らない。**

4つのカテゴリから最低1つずつ検討し、合計3つ以上を生成する:

### カテゴリA: 広告運用要因
- 予算・入札の変更はあったか
- クリエイティブの追加・停止はあったか
- ターゲティングの変更はあったか
- 学習期間中ではないか

### カテゴリB: 外部環境要因
- 季節要因（求職ピーク・閑散期）に該当するか
- 競合の広告出稿変化はあるか
- 業界全体のトレンド変化はあるか

### カテゴリC: 技術的要因
- CVトラッキングは正常に動作しているか
- LP（ランディングページ）のページ速度に問題はないか
- API接続エラーは発生していないか

### カテゴリD: サイト側要因
- LPに変更があったか
- 応募フォームに問題がないか
- コンテンツ（採用コラム等）の公開/削除があったか

### 仮説フォーマット

各仮説を以下の形式で記述する:

```
仮説 #N: [仮説の内容]
- カテゴリ: A/B/C/D
- もし正しいなら: [確認すべき証拠]
- もし間違いなら: [否定する証拠]
- 検証手段: [どのデータソースで確認するか]
```

## Step 3: 体系的な検証

**最初に合致した仮説で止まらない。全仮説を検証する。**

1. 仮説を検証しやすい順（データが取りやすい順）に並べ替える
2. 各仮説について、cross-data-sources.md の検証マトリックスに従いデータを取得する
3. learned-corrections.md の既知パターンに該当しないか照合する
4. 各仮説の検証結果を以下の形式で記録する:

```
仮説 #N: [仮説の内容]
- 検証結果: 支持 / 部分的に支持 / 否定
- 根拠: [確認したデータと結果]
- 確信度: 高 / 中 / 低
```

### 判断ルール

- **単一仮説が高確信度で支持**: それを主因として報告。ただし他の仮説の検証結果も併記する
- **複数仮説が部分的に支持**: 複合要因として報告。寄与度の推定を試みる
- **どの仮説も十分に支持されない**: Step 4（反復）へ進む

## Step 4: 反復

**検証中に新事実が見つかれば、新しい仮説を生成して繰り返す。**

- Step 3で予期しないデータパターンが見つかった場合、新しい仮説を追加する
- 新仮説もStep 3の検証プロセスに従う
- ループ終了条件:
  1. 全仮説の検証が完了し、高確信度の根本原因が特定できた
  2. 利用可能なデータソースを全て確認し、それ以上の深掘りが不可能
  3. 分析をエスカレーションすべきと判断（人間の判断が必要な場合）

## 出力テンプレート

分析結果は以下の構造で報告する:

```markdown
## 分析結果サマリー

### 事実（Step 1）
[主要KPIの変動を箇条書き]

### 仮説と検証（Step 2-3）

| # | 仮説 | カテゴリ | 結果 | 確信度 |
|---|------|---------|------|--------|
| 1 | ... | A | 支持 | 高 |
| 2 | ... | B | 否定 | — |
| 3 | ... | D | 部分的 | 中 |

### 結論
[根本原因の特定結果。複合要因の場合は寄与度を推定]

### 推奨アクション
[結論に基づく次のステップ]

### 注意事項
[季節要因・学習期間等の文脈情報]
```
